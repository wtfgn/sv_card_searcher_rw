# sv_helper
- Search your cards
- Build your own deck

Relies on the api https://github.com/wtfgn/shadowverse_api
## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
